# Security-Portfolio

Personal portfolio site for Obianwu Michael — an aspiring security software engineer. Design inspired by [santifer.io](https://santifer.io/).

## Stack

Next.js 15 (App Router) · React 19 · TypeScript · Tailwind CSS · next-themes · Framer Motion · Lucide

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Edit Content

All content is hardcoded inside the `components/*-section.tsx` files. See [CLAUDE.md](CLAUDE.md) for editing conventions and how to add new projects, certifications, or sections.
